# Jaehai-Webpage

## How to use

### 1. Fork this repository

### 2. Clone the repository

```bash
git clone https://github.com/ops-jaeha/jaeha-webpage.git
```

### 3. Install dependencies

```bash
npm install
```

### 4. Config env.json

- `config/env.json` 파일을 수정합니다.

### 5. Run the development server

```bash
npm run dev
```
